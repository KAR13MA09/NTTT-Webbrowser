## Installing
``` bash
git clone https://github.com/KAR13MA09/NTTT-Webbrowser

cd NTTT-webbrowser

bash install.sh

python NTTT.py
```

### Argument
```
python NTTT.py 
    --start = Start App
    --url <URL> = Search Website
    --help = Help
    --version = Version
```
